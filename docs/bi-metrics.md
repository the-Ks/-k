# BI 看板指标口径

## 客服综合质检平均分趋势

展示统计周期内，每天已完成质检会话的平均综合分。

综合分满分 100，建议由以下维度组成：

- 响应速度：20 分
- 回答专业度：30 分
- 服务态度：20 分
- 流程合规：20 分
- 风险扣分：10 分

计算方式：

```text
某日平均质检分 = 当日已完成质检会话总分之和 / 当日已完成质检会话数
```

页面中同时展示：

- 日期
- 星期
- 当日已质检会话数
- 当日平均质检分
- 当日超时会话占比

## 首次有效响应时长分布

展示客户提出有效问题后，客服首次有效人工回复所用时间的分布。

不计入：

- 系统消息
- 自动回复
- 表情或无意义回复
- 客户连续补充消息导致的重复问题

分布区间：

- 0-1 分钟
- 1-3 分钟
- 3-10 分钟
- 10 分钟以上

## 客户问题类型分布

从客户消息 `content` 中识别问题主题。

一个会话可以命中多个问题类型，例如客户同时问“效果”和“售后”，两个类型都计数。

常见问题类型：

- 产品效果
- 价格优惠
- 售后保障
- 使用方法
- 物流发货

## 客服质检排行

按统计周期内客服已完成质检会话的平均综合分排序。

页面中展示：

- 客服姓名
- 平均质检分
- 已质检会话数
- 平均首次响应时间
- 关联高意向客户数量
